# ITEC3230 - QuickSmile Dental
The QuickSmile Dental website is a front-end web application designed to allow patients and clinic staff to interact with a dental booking system. The purpose of this project is to demonstrate user interface design, user experience flow, and system functionality without any backend.

The system allows:
- Patients to view, create, and manage appointments
- Clinic staff to view and manage all bookings
- Role-based navigation between patient and staff dashboards
